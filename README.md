# ShortRent
o2o房屋租借平台

## 介绍

​	轻松短租网为一个房屋租住平台，用户注册登录后可以创建房屋，待平台管理员审核后可以发布出去，供其他用户预定，同时用户可以根据地点和房屋名称来搜索房屋，查看房屋的详细介绍，并可以下订单预定房屋，当房东受理您的订单后，表示您已成功预定。为房屋发布人提供了出租平台，为房客出行租房提供了便利。 轻松短租网特点：实行线上预定，线下交易。 房客预定房屋时，支付订金，订金由平台保存，房东受理房东的订单。
